# System Architecture

The architecture consists of eight layers:

1. Data Collection Layer
2. Event Storage Layer
3. Digital Behavior Twin Layer
4. UEBA Layer
5. DLP Layer
6. Incident Management Layer
7. Blockchain Evidence Layer
8. AI Investigation Layer

Data Flow:

Log Sources
    ↓
Collection
    ↓
Storage
    ↓
Behavior Twin
    ↓
UEBA
    ↓
DLP
    ↓
Incident
    ↓
Evidence
    ↓
Blockchain
    ↓
Investigation
